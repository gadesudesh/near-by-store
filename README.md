# 🏪 Near By Store

A full-stack web app that connects customers with nearby local shops.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS, Leaflet Maps
- **Backend:** Node.js, Express.js, MongoDB
- **Auth:** JWT

## How to Run

### Prerequisites
- Node.js (v18+)
- MongoDB (local or Atlas)

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/near-by-store.git
cd near-by-store
